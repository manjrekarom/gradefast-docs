Evaluate Module¶

Evaluate module provides Evaluate to execute a GFTest on a SubmissionGroup

class gradefast.evaluate.Evaluate(test, result_path='./', clean=True, timeout=10)[source]¶

Allows evaluating a GFTest on a gradefast.submission.SubmissionGroup.

Parameters

  • test (GFTest) –

  • result_path (str, optional) – Path to persist the ResultGroup as json file. Default is ‘./’

  • clean (bool, optional) – Cleans or resets state of the test working directory after running test on each Submisison

  • timeout (int, optional) – Maximum time to run the test for. Default is 10 seconds

  • todo (.) – Clean option is yet to be implemented
